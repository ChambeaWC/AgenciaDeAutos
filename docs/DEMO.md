# Guion de demo

## Opcion video (recomendada)

Duracion sugerida: 2 a 4 minutos.

1. Presentacion del proyecto e integrantes.
2. Login con usuario administrador.
3. Alta de un vehiculo.
4. Edicion de un vehiculo.
5. Eliminacion de un vehiculo.
6. Alta de un nuevo usuario empleado.
7. Logout.
8. Login con empleado y comprobacion de que no puede entrar a la seccion de usuarios.
9. Cierre con conclusiones.
